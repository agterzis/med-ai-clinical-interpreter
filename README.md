# Med-AI Clinical Interpreter

**MedGemma Impact Challenge 2026 - Kaggle Submission**

Bridging ML predictions and human understanding with Random Forest + MedGemma for cardiovascular risk assessment.

## Overview

Med-AI Clinical Interpreter combines:
- **Random Forest** - Accurate risk prediction with explainable feature importance
- **MedGemma 1.5 4B** - Medical LLM for human-readable reports

The system produces two outputs:
- **Clinical Briefing** - Technical summary for healthcare providers
- **Patient Letter** - Reassuring explanation for patients
